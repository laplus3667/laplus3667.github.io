Process: 處理文本的四道濾鏡

濾鏡一: 句子種類-簡單(Simple), 複合(Compound), 複雜(Complex), 複合複雜(Compound-Complex) (Pragmatics 語用學)

共有四種句子種類(Sentence Variety): 簡單, 複合, 複雜, 複合複雜.

1. 簡單句(Simple):

  • 一個獨立子句(One independent clause): 主部(Subject)+述部(Predicate)
  • 表達一個完整意思

2. 複合句(Compound):

  • 至少兩個獨立子句
  • 利用(1) comma, 對等連接詞(Coordinating conjunction)(2)分號連結起兩個獨立子句
  • 表達「兩個」「完整意思」之間的關係.

3. 複雜句(Complex):

  • 從子句(Subordinate Clause)+主子句(Main clause)
  • 利用「從屬連接詞(Subordinating conjunction)」連結從子句與主子句.

4. 複合複雜(Compound-Complex):

  • 二獨立子句+一相依子句.
  • 盡量少用, 因為是長句子.

濾鏡二: 主述部切割-主部(Subject), 述部(Predicate) (Semantic 語義學)

共有兩大成分: 主部(Subject), 述部(Predicate)

1. 主部(Subject):

  • 與「動詞」無關
  • 包含主詞（名詞、代名詞、片語）和修飾語（形容詞、或與形容詞相等的語詞）。【中文文法把主部稱做「主語」】
  • 主部是句子的主角
  • 主詞是主部的中心
  • 物件屬性(Class attribute) 2. 述部(Predicate):
  • 與「動詞」有關
  • 包含述語動詞（動詞、助動詞），受詞（名詞、代名詞、片語），補語（名詞、代名詞、形容詞、片語、子句），修飾語（形容詞、副詞）等。【中文文法把述部稱做「謂語」】
  • 述部則說明主部的性質或狀態。
  • 述語動詞是述部的中心
  • 物件方法(Class method).

濾鏡三之一: 主部分析 (Syntax 句法學)

主部: 主詞（名詞、代名詞、片語）+ 修飾語（形容詞、或與形容詞相等的語詞）

濾鏡三之二: 述部分析 (Syntax 句法學)

述部: 述語動詞（動詞、助動詞）+ 受詞（名詞、代名詞、片語）+ 補語（名詞、代名詞、形容詞、片語、子句）+ 修飾語（形容詞、副詞）

受詞:直接與間接:

  • 在述部中，用來接受動詞之動作的對象，稱為受詞【中文文法稱做「賓語」】。
  • 其中直接接受動作的，稱為直接受詞；
  • 間接接受動作的，稱為間接受詞。

補語: 主詞與受詞:

  • 在述部中，用來補充說明主詞或受詞之情況的語詞，稱為補語。
  • 其中補充說明主詞的，稱為主詞補語；補充說明受詞的，稱為受詞補語。

濾鏡四: 片語分析: 名詞片語(Noun Phrase) (詞法學 Morphology)

八大片語: 名詞片語, 動詞片語, 形容詞片語, 副詞片語, 介系詞片語, 分詞片語, 動名詞片語, 不定詞片語

1. 名詞片語(Noun Phrase):

Noun Phrase: Noun + Article + Adjective + Prepositional Phrase + Determiner

2. 動詞片語(Verb Phrase):

  • 用helping verb改變語氣
  • 用tense改變relative time
  • 用voice改變主動被動

3. 形容詞片語(Adjective Phrase):

  • 形容主語(補語功能)
  • 形容賓語(補語功能)
  • 數詞是一種特殊的主語, 也可以用形容詞片語修飾

4. 副詞片語(Adverb Phrase):

  • 修飾動詞.
  • 在英語中, 形式上很像介系詞片語.
  • 功能多樣, 可以回答when, where, why, how. 不能回答who,what.
  • 就近修飾原則: 不同位置有不同修飾功能.

5. 介系詞片語(Prepositional Phrase):

  • 格退化後的產物: by(五格), at(六格), for(三格), in(六格), on(六格).
  • 根據修飾功能, 可能是形容詞片語(修飾名詞), 也可能是副詞片語(修飾動詞).

6. 分詞片語(Participle Phrase):

  • 關係代名詞縮減版: 修辭目的, 把關係代名詞省略掉.
  • 現在分詞修飾「主動語氣」中的主語:主動動詞轉分詞片語, 做形容詞功能修飾主語
  • 過去分詞修飾「被動語氣」中的主語：被動動詞轉分詞片語, 做形容詞功能修飾主語

7. 動名詞片語(Gerund Phrase):

  • 動詞轉品, 成為主語: 當主語要是「動作」, 需要將動詞轉為「動名詞片語」來當作主語.
  • 名詞行為: 動名詞片語的行為同名詞, 可以做「主詞」, 「直接受詞」, 「間接受詞」三者作用. 更進階也可當作「謂語名詞」, 「同位語」使用.

8. 不定詞片語(Infinitive Phrase):

  • 補充意思不完整的動詞.
  • 動詞可以加的賓語, 修飾成分, 不定詞片語都可以繼承
  • 可以做形容詞, 修飾名詞(表作用, 動作)
  • 也可以做副詞, 修飾動詞(表目的, 潛台詞)

Output: 輸出可重用備忘- Source-模式1->Clause-模式2->Expression-模式3->Case Phrase-模式4->Collocation.

  1. 文本–>去掉子句轉接–>子句群
  2. 子句–>主部(Subject);述部(Predicate)–>兩種表達
  3. 主部, 述部–>格理論–>六格片語
  4. 六格片語–> 格還原–> 搭配詞(Collocation)

  • 句子種類:Sentence Variety:{Sentence}–>{Simple, Compound, Complex, Compund-Complex}.

輸出模式一: Source –「句子類型, 連接詞」–> Clause

4種tags: #Simple;#Compound;#Complex; #CompoundComplex

連接詞應該就不用tags了.

Metadata 1: Sentence Variety: 四種句子類型. Metadata 2: Coordination: {If not simple sentence}, 對等連接詞, 從屬連結詞. Metadata 3: Sentence Modifier: 也有可能有修飾整個子句的成分, 在這個模式要抓出來去掉. Output: Clause list: 可以有1,2,3個Clause

輸出模式二: Clause – 「主述部切割」–> {Subject Expression, Predicate Expression}

兩種Tags: #SubjectExpression; #PredicateExpression

Metadata 1: Subject Expression: 主部, 包含主語, 主語修飾成分

Metadata 2: Predicate Expression: 述部, 包含動詞需要的搭配, 修飾動詞的副詞.

輸出模式三: {Subject Expression, Predicate Expression} – 「格理論」–>{Case Phrase}

六種tags: #Num; #Gen; #Dat; #Acc; #Ins; #Prep

主部都是主格, 直接格標記Nominative

述部的成分可能有五格, 需要做五格標記

Metadata 1: Nom: {Subject Expression}–>{Nominative Case}

Metadata 2: Gen: {Predicate Expression}–>{Genitive Case}

Metadata 3: Dat: {Predicate Expression}–>{Dative Case}

Metadata 4: Acc: {Predicate Expression}–>{Accusitive Case}

Metadata 5: Ins: {Predicate Expression}–>{Instrumental Case}

  • 方法手段

Metadata 6: Prep: {Predicate Expression}–>{ Prepositional Case}

可否用六格理論(Case Theory)來拆解主述部?:

  • 主格(Nominative;被給的名稱)
  • 屬格(Genitive;與生俱來的)
  • 與格(Dative;給與)
  • 賓格(Accusitive;被影響的)
  • 工具格(Instrumental;完成行動所憑藉的工具方法)
  • 前置格(Prepositional;時空關係)

可否用句法形態配列(Morphosyntactic alignment)來拆解主述部?:

  • 施事論元(A;Agent)
  • 受事論元(P;Patient)
  • 變元(S;Subject)

分解語言的句法型態配列:

句法型態配列 定義 實例
主賓型 (Nominative–accusative alignment) 主格=A,S; 賓格=P 英語, 日語, 世界上大多數語言
中立型 (Neutral) A,P,S 三者無格的區分 漢語

後記

寫了150分鐘, 感受到體系的龐大！之後還需要花些精力研究具體的phrase構造, 主部述部的各種細節. 走通後可以建立一套metadata系統, 就可以在Obsidian建立有用的template來標準化文件輸出.

繼續寫了100分鐘, 感受到格理論的好用之處, 在主述部切割以後, 進行格標記, 來把句子拆做六大格片語; 利用格片語, 再來與單字連結, 實踐最小意義單位. 感覺系統已經workable了！實踐到句子分析上玩玩看.

實踐在Obsidian上面感覺不錯！持續改善系統, 可以scaling the language learning progress!
